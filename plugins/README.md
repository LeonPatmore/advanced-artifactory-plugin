- [Advanced Spring Boot Service Plugin](advanced-spring-boot-service-plugin)

## Usage

### Local

To set up a local artifactory instance:

`make local`

To push the plugins to this instance:

`make publish`

### UI

http://localhost:8080/artifactory/webapp/home.html
